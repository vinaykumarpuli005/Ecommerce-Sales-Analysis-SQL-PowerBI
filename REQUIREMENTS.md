# Requirements

## Software Required
- SQL Server
- SQL Server Management Studio
- Power BI Desktop
- GitHub account

## Dataset Required
- Online Retail Dataset from UCI Machine Learning Repository
- Convert the dataset to CSV format before importing into SQL Server

## Skills Used
- SQL data cleaning
- SQL Server table creation
- SQL views
- Aggregate functions
- Data filtering
- Power BI dashboard design
- DAX measures
- Business analysis
- Data visualization

## Power BI Measures Used
- Total Revenue
- Total Orders
- Total Customers
- Average Order Value
- Total Quantity Sold

## Notes
Before running the SQL file, update the CSV file path in the `BULK INSERT` query based on your local system.

The Power BI dashboard was built using cleaned SQL data. If the dashboard does not refresh, update the SQL Server connection in Power BI.
